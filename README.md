# STEMFlow AI 🚀

STEMFlow AI is an AI-powered STEM learning platform designed to help students understand complex concepts through interactive lessons, quizzes, exams, and personalized feedback.

Built with React, Next.js API routes, OpenRouter, and DeepSeek AI.

## ✨ Features

### 📚 AI Tutoring

* Ask questions about Science, Technology, Engineering, and Mathematics.
* Structured explanations with clean markdown formatting.
* Focus on understanding rather than memorization.
* Context-aware conversations.

### 📝 Quiz Generation

* Automatically generates quizzes from AI-generated lessons.
* Includes:

  * Multiple Choice Questions (MCQs)
  * Short Answer Questions
* Instant AI-powered grading.

### 🎯 Exam Generation

* Generate complete exams from lesson content.
* Mixed difficulty levels:

  * Easy
  * Medium
  * Hard
* Multiple question formats:

  * MCQ
  * Short Answer
  * Long Answer

### 🤖 AI Assessment

* Intelligent grading system.
* Supports partial credit.
* Accepts equivalent wording.
* Provides detailed feedback and explanations.

### 📱 Responsive Design

* Desktop optimized.
* Mobile friendly.
* Auto-scrolling chat experience.
* Modern dark UI.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/abdullah-butt-dev/stemflow.git
cd stemflow
```

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env.local` file:

```env
OPENROUTER_API_KEY=your_api_key_here
```

### Run Development Server

```bash
npm run dev
```
